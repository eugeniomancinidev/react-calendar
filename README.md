# React Calendar

Interactive calendar built with React using functional components and hooks.

## Live Demo

https://react-calendar-e3w1jnqgy-eugenio-mancinis-projects.vercel.app

## Features

- Month and year navigation
- Week starting from Monday
- Correct weekday alignment
- Highlight of the current day
- Clean and responsive UI
- Component-based architecture

## Tech Stack

- React
- Vite
- CSS

## Run Locally

```bash
npm install
npm run dev
